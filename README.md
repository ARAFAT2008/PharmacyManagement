# PharmacyManagement
A Pharmacy Management System created as a first-semester university project, demonstrating basic concepts of inventory and sales management.
The Data folder contains information on over 25,000 medicines listed on MedEx.

Steps to Prepare the Project
make sure logo.txt and data.7zip is in the same folder.
Extract data.7z into the same directory as main.c.
